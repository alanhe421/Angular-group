## 如何提问
- 大家都很忙，没有谁有义务和责任回答你的问题。so,多点耐心，多点理解，也多点修养。
- 想让别人快速了解你问的，最好的方式是代码，建议[jsbin](https://jsbin.com)创建一个demo，那样快速复现问题。同时也是锻炼你表达问题，思考问题的能力。你要知道会问问题也是本领。
